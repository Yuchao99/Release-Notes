Lightcloud Release Notes: August, 2 2017

Today’s release includes a host of new hardware options and software features.

  • Lightcloud™ Sensor Updates
  • Multi-Zone Dimming
  • Configurable Fade Times
  • Power-Up State
  • Two-Factor Authentication
  • New Hardware Colors

Lightcloud Sensor Updates

With today’s update, we’re excited to announce three big updates to the Lightcloud Sensor.

1. The Lightcloud Sensor now features dual technology! Noise detection is used to complement its Passive Infrared (PIR) sensor, increasing accuracy when detecting vacancy.

  • This dual technology update is available for both new AND existing Sensors.

2. The Sensor is available with new mounting accessories. All Sensors are available with three mounting options:

  • Ceiling / surface-mount (the standard version available since launch)
  • Fixture/Junction Box (1/2” NPT)
  • Fixture/Junction Box (1/2” NPT) with height adjuster

3. The Lightcloud Sensor now comes in a high bay version that can be mounted at heights up to 48 feet.

To summarize the new options and catalog numbers:

Multi-Zone Dimming

  • Multi-Zone Dimming allows a user to dim up to 15 zones with a single Lightcloud Dimmer switch. This is an example of multi-zone dimming configuration using the ‘control zones’ settings.

Configurable Fade Times

  • With Configurable Fade Times, users can now set custom fade times for different Zones in a Scene.

Power-Up State

  • The new Power-Up State feature allows users to tell a Lightcloud device what to do when power is restored after an outage. (This is configurable from Device Settings.)

    There are three options are available:

    1. Last state (immediately before power lost)

    2. Last on level (on, at last dim level before power lost)

    3. Custom level selected by the user

Two-Factor Authentication

  • Two-Factor Authentication provides an optional layer of additional security. When enabled, a user receives a text message with a verification code that must be entered when logging in.

New Hardware Colors

  • The Lightcloud Controller and Touch are now available in white, making them perfect for use with white fixtures, ceilings and walls.
