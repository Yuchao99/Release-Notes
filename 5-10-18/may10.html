Lightcloud Release Notes: May, 10 2018

Today’s launch is full of powerful new features:

  • Fault Notifications
  • Grouped Sensor
  • Closed Loop Daylighting
  • Faster Power Up for Controller

Fault Notifications

Lightcloud can send you an email alert whenever a Zone is experiencing irregular energy consumption. Now, each Zone can be configured for Fault Notification with adjustable thresholds – perfect for a Device controlling multiple fixtures. Configuration instructions below.

Grouped Sensors

Sensors can now be grouped together to act like a single Sensor. For example, if a large room has 4 Sensors, Group Vacancy will keep all of the lights on until all of the Sensors register vacant. Configuration instructions below.

Closed Loop Daylighting

Lightcloud is now capable of closed loop Daylight harvesting on Standard and High Bay Sensors.
Closed loop Daylighting optimizes the light level in a location by dimming one or more luminaires to account for natural and artificial light in a space. When there’s a lot of natural light in a space, the lights are dimmed until the optimal light level is reached. Configuration instructions below.

Faster Power Up for Controller

Controllers now power up 10x faster than before.

Configuration Instructions

Fault Notifications

Set an Alert Threshold in Device Settings.
Please note, the alert Threshold is the % change from normal consumption and requires Power Monitoring to be turned on.

After setting the Alert Threshold, turn on fault detection.

To receive alerts, one or more users or emails needs to be subscribed. To configure alerts, go to Fault Notifications under the settings menu and select users or enter email addresses to receive alerts. Click on Select Sites.

Select the Site or Sites the user is subscribing and press Done.

The top of the page will show all subscribed users. To unsubscribe a user, select Delete.

Grouped Sensors

Select a Sensor and select Browse next to Browse All Ceiling Sensor Automations.

Find the vacancy automation that you’d like to group with other Sensors and select Devices.

Select the Sensors you’d like to group and Done. Select Save to preserve the new Grouped Vacancy.

Closed Loop Daylighting

Select Add next to Add Daylighting.

Select the footcandles you’d like to maintain, Add Zones, and Save.
