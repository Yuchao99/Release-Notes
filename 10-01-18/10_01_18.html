Lightcloud Release Notes: Octorber, 01 2018

Today’s launch is full of powerful new features:

  • Multiple PINs
  • Faster Controls
  • User Permissions
  • Real-Time Top Bar
  • Optimized Time Zones
  • Luminaire Controller

Multiple PINs

The Touch is now capable of multi-user PINs for user permissions and user logging.

Faster Controls

Communication protocols have been refined for noticeably faster on-site controls.

User Permissions

User permissions and settings have a new look that’s easier to use.

Real-Time Top Bar

The join mode top bar now has real-time synchronization.

Optimized Time Zones

Remote time zones scheduling has been optimized for precise time-based controls across North America.

Luminaire Controller

Additional support has been developed for the all new Luminaire Controller.
Learn more about the Luminaire Controller.

System Optimizations

Lightcloud is constantly getting better through regular bug fixes and optimizations.

Configuration Instructions

Multiple PINs

Set the Passcode Mode to “Multiple” under the Touch’s Device Settings and select Save.

Set Touch PIN Enabled to Enabled under the User Settings and select Save.

User Permissions

Select the Areas or Zones you’d like to grant Control or Edit rights under User Settings.
